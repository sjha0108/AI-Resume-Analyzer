# AI-Powered Resume Analyzer

An intelligent web app that analyzes resumes using AI and scrapes LinkedIn for jobs.

## Features
- Resume Summary
- Strength Analysis  
- Weakness Analysis
- Job Title Suggestions
- LinkedIn Job Scraper

## Tech Stack
- Python, Streamlit, Groq AI, Selenium, PyPDF2

## How to Run
pip install -r requirements.txt
streamlit run app.py
